The Diagonalization Argument

How did I get here?

During that time I usually earned some extra money as a tutor for theoretical CS and mathematics.

Back then I often observed that both students and teachers, struggled to explain the technique of the diagonal argument in a general sense.

Usually, most of the students were able to grasp the individual applications of that technique, Cantors uncountability proof for \(\mathbb{R}_{[0,1]}\), or the power set theorem referring to \(|\mathcal{P}(S)| > |S|\) - or the halting problem - depending on the field of study of the respective student or teacher.

TL;DR

What I would like to show here on a more formal level, is what the diagonal argument actually is, and how to use it in its usual or contrapositional form.

The Diagonal Argument

Given a function \(f: I \rightarrow S\), a function \(P: I \times S \rightarrow T \), which is more like a property, i.e. \(|T| \geq 2\) is the only necessary condition for \(P\). The following holds

\begin{gather} \exists s^* \in S\ \forall i \in I: P(i, f(i)) \neq P(i, s^*) \\ \text{implies }f\ \text{ is not surjective.} \end{gather}

It might look much more complicated than it actually is. If $f$ is surjective, then $S$ is the range of $f$, $S = f(I)$, and because of $s \in S$ some index $i^*$ would then yield $f(i^*) = s^*$, thus $P(i^*, f(i^*)) = P(i^*, s^*)$ contradicting the construction of $P$.

The equivalent, contrapositional form is:

\begin{gather} \text{If } f \text{ is surjective, then:}\\ \quad \not\exists s^* \in S\ \forall i \in I: P(i, f(i)) \neq P(i, s). \end{gather}

We could formally negate the implication (for all \(s^*\) exists an \(i\) sich that …) - but that is not the way the diagonal argument is usually applied. Rather that non-existence is exploited to refute a certain construction of $s^*$… which would also be my meta-theorem-like takeaway from the formal lighting of the diagonal argument:

The two forms which I am aware of are:

  1. The direct form - the goal is to prove that no surjective \(f\) exists.
    You are able to construct that special element \(s^*\) which is not in the image of \(f\) by exploiting the inner structure of the set \(S\) and the property \(P\) - the P-in-equality usually provides the means to do so.

  2. The contrapositional form - the goal is to contradict some construction for \(s^*\):
    You already know, by some theorem, that a surjective \(f\) exists. A construction of \(s^*\) then involves several steps that are proven to be legitimate and one step which is, only by assumption \(A\), legal/existent. By showing that \(\forall i \in I: P(i, f(i)) \neq P(i, s^*)\) for the constructed/hypthetical \(s^*\), you have contradicted \(f\) being surjective and thus proven the construction to be invalid, and thus the assumption \(A\) to be wrong.

    1. The restriction that, in the constructino of \(s^*\), only one assumption \(A\) is made, is crucial. If several steps in the construction of \(s^*\) are questionable, your proven result is weak in the sense that your proof doesn’t show anymore, precisely, which step in the construction fails.

Fun fact: As in any theorem we might use it to falsify any aspect if it by just making clear that every other aspect of our proof / construction is valid. It might be thinkable to use the diagonal argument to prove that \(|T| = 1\), but I can’t think of an application like that. If you are aware of such an example, please let me know (but please give me some relevant citation, not an artificial, made-up example). Also, the coincidence that $I$ is part of the domain of $f$ as well as $P$ is, in my opinion, a rather ‘practical’ destillate of its usual applications - if preferred, a distinction could be made, see (**).

The classic examples of how to apply the diagonal argument

Cantors uncountability proof for \(\mathbb{R}_{[0,1]}\)

We want to prove that the set of real numbers in the closed interval $[0,1]$ is uncountable. I.e. that any enumeration $f: \mathbb{N} \rightarrow \mathbb{R}_{[0,1]}$ is not surjective.

We are going to first prove that any enumeration into $\{0,1\}^N$, the set of infinite 0-1-sequences, is not surjective, and are then going to show that $\{0,1\}^N$ could be embedded (injective) into $\mathbb{R}_{[0,1]}$.

  1. First step: showing $|\mathbb{N}| < |\{0,1\}^N|$
  2. Second step: showing $|\{0,1\}^N| \leq |\mathbb{R}_{[0,1]}|$
    1. Thus obtaining $|\mathbb{N}| < |\mathbb{R}_{[0,1]}|$

Proof:
Suppose we got an arbitrary surjective enumeration $f: \mathbb{N} \rightarrow [0,1]^N $

For the reader, we relate to the symbols used in the diagonal theorem above:

  • \(I \equiv \mathbb{N}\)
  • \(S \equiv \{0,1\}^N\)
  • \(T \equiv \{0,1\}\)

The property \(P\) is defined by the i-th component of an element $s \in S$.

\begin{align} P(i, s) = s_i\begin{cases} 1\\ 0 \end{cases} \end{align}

We see now, because the codomain $P$ is two-valued, we can always define a different i-th component for our exceptional sequence $s^*$:

\begin{align} s^*_i = 1 :\iff 0 = P(i,f(i)) = s_i \\ s^*_i = 0 :\iff 1 = P(i,f(i)) = s_i \\ \end{align}

Application of the diagonal argument then yields that $f$ is not surjective, $f(\mathbb{N}) \subsetneq \{0,1\}^N$. Because $f$ was arbitrary, we have shown that $N < [0,1]^N$

While a continued fraction like $0,111111... = 1$ in binary, it is not in decimal representation, which will be the aforementioned injective embedding:

Viewing the sequences in $\{0,1\}^N$ as decimal fractions, the first digit left of the comma always being 0, and leaving aside all other digits from 2 to 9, we have an injective embedding of $\{0,1\}^N$ into $\mathbb{R}_{[0,1]}$ (thank you wikipedia).

That finally yields the second inequality $\{0,1\}^N \leq \mathbb{R}_{[0,1]}$, so $|N| < |\{0,1\}^N| \leq |\mathbb{R}_{[0,1]}|$. $\blacksquare$

Cantors theorem regarding the cardinality of power sets

The power set of a set $X$ is the set of all subsets of $X$, denoted by $\mathcal{P}(X)$.

As long as $X$ is finite, $|\mathcal{P}(X)| = 2^{|X|}$, which is kind of clear and could be understood with a few examples, but for infinite sets intuition might fail.

Again, trying to do good teaching:

  • Let $f$ be some mapping from $X$ to $\mathcal{P}(X)$.
  • We assume $f$ is onto $\mathcal{P}(X)$, and will refute just that by using the diagonal argument.
  • $ I \equiv X $
  • $ S \equiv \mathcal{P}(X)$
    • We will denote elements of $X$ by $x$, and elements of $\mathcal{P}(X)$ by $\underline{x}$
  • $ T \equiv \{true,false\} $

The property \(P\) is then defined via:

\begin{align} P(x, \underline{x}) := x \in \underline{x} \end{align}

That is, given some element $x \in X$, and some subset of $X$, which is by itself an element of $\mathcal{P}(X)$ - that is $\underline{x} \in \mathcal{P}(X)$ - the property $P$ is true if $x$ is an element of $\underline{x}$ and false otherwise.

This hints a way to choose membership for all $x \in X$, and membership is what defines a subset of $X$, thus an element of $\mathcal{P}(X)$.

We define our special element, $\underline{x}^*$, as follows:

\begin{align} \forall x \in X: x \in \underline{x}^* :\iff x \notin f(x) \end{align}

But that boils down to:

\begin{align} \forall x \in X: P(x, \underline{x}^*) \neq P(x, f(x)) \end{align}

Which is, by the diagonal argument, a contradiction to the assumption that $f$ is surjective. $\blacksquare$

The halting problem

This topic is treated a little less formal (***), but doing differently would be beyond scope.

The halting problem is the problem of determining, given a program and an input, if that program will halt and yield some result, or run forever.

We will call such a fictional program the test, and it would be tremendously useful to have such a program, because we could abort program execution before ‘accidentally’ running into an infinite loop.

Thinking about this from a philosophical perspective will soon run yield contradictions, and for me this is part of my fascination for theoretical computer science, especially since the non-existence of such a test isn’t mere ‘philosophy’, it is just not existent.

This has applications in static program analysis (e.g. type inference, but was relevant in understanding the PCP, and gives bounds on how far we can determine compliance of a program given a certain specification.

One might argue that this is all pretty old stuff, but so is a lot of ‘pure’ theory 1.

The basic knowledge needed to follow this example is that all programs and also their inputs could be binary encoded, and that binary encoding boils down to $\mathbb{N}$.

So, again, to connect to the inital symbols of the diagonal argument - this time used in its contrapositional form:

  • $A$ is the assumption that a test program $s'$ exists.
  • $I \equiv \mathbb{N}$, the set of all inputs
  • $S \equiv \mathbb{N}$, the set of all programs (**).
  • $f$ is the identity function on $\mathbb{N}$, $f(i)=i$, so we won’t explicitly mention it.
  • $T \equiv \{halt,loop\}$.
    • $P(i,s)$ is the property that program $s$ terminates on input $i$.
  • N.B.: we know that $f$ is surjective.

It is important to understand that the assumption $A$ does not contradict the existence of $P$, which is a property of any program that holds or does not hold - no questions asked / no relevance about how to algorithmically determine that. It is the existence of an actual program $s'$ that does just that, which is in question.

Using the assumption $A$, the existence of a halting test $s'$, we could construct a program $s^*$ that behaves different with regards to $P$ than any other program $s$.

That is, we call $s'$ as a ‘subroutine’, which is the informal part (***) I mentioned earlier, and then decide to loop forever if the test yields “halt”, and we halt if the test yields “loop forever” - something like that, assuming strings are just binary input and could be mapped to natural numbers:

func s_star(input string) string {
  if halt_test(input) == "halt" {
    for {
      // looping forever
    }
  } else {
    return "halt"
  }
}

Mathematically, we would express that as:

\begin{align} P(i, s^*) = \begin{cases} halt & \text{if } P(i,i) = loop \\ loop & \text{if } P(i,i) = halt \end{cases} \end{align}

So, again, by construction:

\begin{align} \forall i \in I: P(i, s^*) \neq P(i, i) \end{align}

So, by the diagonalization argument, $f$ would not be surjective - but it is, which contradicts the assumption $A$, the existence of a test program $s'$ which we used in the construction of $s^*$. $\blacksquare$
